## 🇬🇧 English

**GeolyX** is a sleek desktop application that generates Google UULE (Unified Location/URL Encoder) codes in seconds. Covering all **81 provinces and hundreds of districts** of Turkey, it lets you copy the UULE parameter for location-based Google search with a single click.

### ✨ Features

- 🗺️ **81 provinces**, hundreds of districts — full Turkey coverage
- 🎯 **One-click copy** — both raw UULE and `&uule=` URL-ready format
- 🎨 **Minimal, elegant** — warm earth-toned UI with smooth interactions
- ⚡ **Fully offline** — no internet required, all data embedded
- 📋 **Auto-clipboard** — copy directly to your clipboard

### 🚀 Usage

1. Select a **Province** (İl)
2. Select a **District** (İlçe)
3. Click **Kopyala** (Copy) — your UULE code is ready

```
Example output:
  Raw:      w+CAIQICInS2F5c2VyaSxLYXlzZXJpLVR1cmtpeWU=
  With URL: &uule=w+CAIQICInS2F5c2VyaSxLYXlzZXJpLVR1cmtpeWU=
```

### 🔧 Requirements

- Python 3.8+
- Windows (uses `ctypes` for taskbar integration)
- Tkinter (included with Python)

## 🇹🇷 Türkçe

**GeolyX**, Google UULE kodlarını saniyeler içinde oluşturan şık bir masaüstü uygulamasıdır. Türkiye'nin **81 ilini ve yüzlerce ilçesini** kapsayan veritabanıyla, Google'da konum bazlı arama yapmak için gereken UULE parametresini tek tıkla panonuza kopyalayın.

### ✨ Özellikler

- 🗺️ **81 il**, yüzlerce ilçe — Türkiye'nin tamamı
- 🎯 **Tek tıkla kopyala** — hem ham UULE hem de `&uule=` URL formatı
- 🎨 **Minimal ve zarif** — sıcak toprak tonlarında akıcı arayüz
- ⚡ **Tamamen çevrimdışı** — internet gerektirmez, tüm veri gömülü
- 📋 **Otomatik pano** — doğrudan panoya kopyalama

### 🚀 Kullanım

1. Bir **İl** seçin
2. Bir **İlçe** seçin
3. **Kopyala** butonuna tıklayın — UULE kodunuz hazır

```
Örnek çıktı:
  Ham:        w+CAIQICInS2F5c2VyaSxLYXlzZXJpLVR1cmtpeWU=
  URL ile:    &uule=w+CAIQICInS2F5c2VyaSxLYXlzZXJpLVR1cmtpeWU=
```

### 🔧 Gereksinimler

- Python 3.8+
- Windows (görev çubuğu entegrasyonu için `ctypes` kullanır)
- Tkinter (Python ile gelir)
